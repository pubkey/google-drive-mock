# google-drive-mock
![google-drive-mock](google-drive-mock.png)

Mock-Server that simulates being google-drive. Used for testing the [RxDB Google-Drive-Sync](https://rxdb.info/). Mostly Vibe-Coded.



## Tech

- TypeScript
- Express
- Vitest
